# Algortm
Get a list and return all possible combinations of that list.

Example 1:
- Input: `[1, 2, 3]`
- Output: `[[], [1], [2], [1, 2], [3], [1, 3], [2, 3], [1, 2, 3]]`

Example 2:
- Input: `[0]`
- Output: `[[], [0]]`

Example 3:
- Input: `[1, 2, 3, 4]`
- Output: `[1, 2], [3], [1, 3], [2, 3], [1, 2, 3], [4], [1, 4], [2, 4], [1, 2, 4], [3, 4], [1, 3, 4], [2, 3, 4], [1, 2, 3, 4]]`
